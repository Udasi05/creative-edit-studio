# Creative Edit Studio

Professional video editing portfolio website.

## Project info

A showcase of professional video editing services including color grading, motion graphics, and cinematic editing.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd creative-edit-studio

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
